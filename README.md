Developed an interactive Adidas Sales Dashboard using Power BI to analyze sales performance across regions, states, products, and retailers.

Key features:
• Total Sales, Profit, Units Sold, Average Price & Margin KPIs
• Monthly sales trend analysis
• Regional and state-level sales visualization using maps
• Product-wise and retailer-wise profit breakdown
• Dynamic filters for region and invoice date

Tools Used: Power BI, DAX, Power Query
